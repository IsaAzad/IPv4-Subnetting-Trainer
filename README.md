IPv4 Subnetting Trainer
An interactive browser-based IPv4 subnetting practice tool.

This lightweight web application generates random IPv4 addresses with CIDR notation and allows users to manually calculate and test subnet details in real time.

Features
Random IPv4 + CIDR generator
Manual subnet calculation practice
Instant answer validation with per-field feedback
Score tracking (attempts, perfect solves, last result)
Reveal full answer key
Handles special cases (/31 and /32)
Fully client-side (no backend required)
What You Practice
For each generated IP/CIDR, calculate:

Network ID
First Host
Last Host
Broadcast Address
Next Subnet
Subnet Mask (dotted decimal)
Total number of hosts
Number of usable hosts
Technologies Used
HTML5
CSS3
Vanilla JavaScript
No frameworks. No dependencies. Runs locally in any modern browser.

How to Use
Download or clone the repository.
Open ipv4_subnet_trainer in your browser.
Click Random to generate a subnetting challenge.
Enter your answers and click Check Answers.
Purpose
Built to improve subnetting speed, accuracy, and conceptual understanding for networking and cybersecurity students.
